## Getting Started

### Prerequisite
install all tect stack 

  ```sh
  npm i
  ```

### Installation
1. Clone this repo
   ```sh
   git clone https://github.com/mrsyaban/image-segmentation
   ```
2. type `npm i` on this directory 
    ```sh
   npm i
   ```
3. then type `npm run dev` 
    ```sh
   npm run dev
   ```
